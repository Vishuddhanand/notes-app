## About
This project is a **full stack Notes Application** developed using the **MERN stack**.  
It demonstrates seamless integration between a **React frontend** and a **Node.js + Express backend**, with **MongoDB** used for persistent data storage.

The application focuses on simplicity, scalability, and clean architecture.
